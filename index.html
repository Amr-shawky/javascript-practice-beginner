JavaScript Assignment Questions

Question 1

Write a program that allows the user to enter a number and then prints it.

Example: Input: 5, Output: 5

Question 2

Write a program that takes a number from the user then prints "yes" if that number can be divided by 3 and 4, otherwise prints "no".

Example 1: Input: 12, Output: Yes

Example 2: Input: 9, Output: No

Question 3

Write a program that allows the user to insert 2 integers then prints the max.

Example 1: Input: 3 5, Output: 5

Example 2: Input: 10 7, Output: 10

Question 4

Write a program that allows the user to insert an integer then prints "negative" if it is a negative number, otherwise prints "positive".

Example 1: Input: -5, Output: negative

Example 2: Input: 10, Output: positive

Question 5

Write a program that takes 3 integers from the user then prints the max element and the min element.

Example 1: Input: 7, 8, 5, Output: max element = 8, min element = 5

Example 2: Input: 3, 6, 9, Output: Max element = 9, Min element = 3
